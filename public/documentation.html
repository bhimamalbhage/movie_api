Movie API Documentation

Welcome to the Movie API! This API provides endpoints for managing movies, users, and favorite movies.

Endpoints

Request URL Response Description
GET /movies Returns a list of the top 10 movies with their titles and release years. Returns a JSON array containing the top 10 movies, each with the title and year of release.
GET /movies/:name Returns detailed information about a movie by title. Returns a JSON object containing details about a specific movie, including title, description, genre, director, image URL, and whether it's featured.
GET /genres/:name Returns details about a specific genre. Returns a JSON object containing details about the genre, such as name and description. Replace :name with the genre’s name.
GET /directors/:name Returns details about a specific director. Returns a JSON object containing details about the director, such as name, birthdate, and filmography. Replace :name with the director’s name.
POST /users Registers a new user in the system. Accepts a JSON object in the request body to register a new user, including fields like username, password, and email.
PUT /users/:username Updates the username of an existing user. Allows updating a user's username. Replace :username with the current username and include the new username in the request body.
POST /users/:username/favorites/:movieId Adds a movie to the user's list of favorite movies. Allows adding a movie to a user's favorite list. Replace :username with the user's name and include the movie details in the request body.
DELETE /users/:username/favorites/:movieId Removes a movie from the user's list of favorite movies. Removes a movie from the user's favorite list. Replace :username with the user's name and :movieId with the id of the movie to be removed.
DELETE /users/:username Deregisters an existing user from the system. Deletes a user account. Replace :username with the user's name to deregister.
