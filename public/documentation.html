Movie API Documentation

Welcome to the Movie API! This API provides endpoints for managing movies, users, and favorite movies.

Endpoints

Request URL Description Request Body (Example) Response (Example)
GET /movies Returns a list of the top 10 movies, including titles and release years. N/A 
[
    { "title": "The Silence of the Lambs", "year": 1991, "genre": "Thriller", "description": "Psychological thriller" },
    ...
]
GET /movies/:name Returns detailed information about a movie by title. N/A 
{
    "title": "The Silence of the Lambs",
    "year": 1991,
    "genre": "Thriller",
    "description": "Psychological thriller",
    "director": "Jonathan Demme"
}
GET /genres/:name Returns details about a specific genre, such as name and description. N/A 
{
    "name": "Thriller",
    "description": "Genre focused on suspense and excitement."
}
GET /directors/:name Returns details about a specific director, including name, bio, birth year, and death year. N/A 
{
    "name": "Christopher Nolan",
    "bio": "Known for complex narratives and unique visual styles.",
    "birthYear": 1970,
    "deathYear": null
}
POST /users Registers a new user in the system. 
{
    "username": "newUser",
    "password": "password123",
    "email": "newuser@example.com"
}
 
{
    "_id": "12345",
    "username": "newUser",
    "email": "newuser@example.com",
    "favoriteMovies": []
}
PUT /users/:username Updates the details of an existing user. 
{
    "username": "updatedUser",
    "email": "updated@example.com"
}
 
{
    "_id": "12345",
    "username": "updatedUser",
    "email": "updated@example.com",
    "favoriteMovies": []
}
POST /users/:username/favorites/:movieId Adds a movie to the user's list of favorite movies. N/A 
{
    "_id": "12345",
    "username": "updatedUser",
    "favoriteMovies": ["movieId123", ...]
}
DELETE /users/:username/favorites/:movieId Removes a movie from the user's list of favorite movies. N/A 
{
    "_id": "12345",
    "username": "updatedUser",
    "favoriteMovies": []
}
DELETE /users/:username Deregisters an existing user from the system. N/A 
User deregistered successfully.
